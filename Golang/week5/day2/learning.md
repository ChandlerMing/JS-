# Week5 Day2 学习内容（超详细指导）

## 今日主题
- JSON 解码

## 学前准备
- 掌握基础 handler。
- 工作目录：`Golang/week5/day2`
- 先执行：`go run template.go`

## 分钟级安排（建议 120 分钟）
- 0-15 分钟：快速阅读模板，标记看不懂的点
- 15-40 分钟：学习核心知识，写下 3 条笔记
- 40-85 分钟：按步骤改代码并验证输出
- 85-105 分钟：做扩展练习
- 105-120 分钟：复盘和整理提交物

## 核心知识讲解
- `json.Decoder` 解码请求体。
- 非法 JSON 返回 400。
- 响应 Content-Type 建议统一。

## 手把手实操（命令级）
1. 打开当前目录下 `template.go`
2. 运行：`go run template.go`
3. 按下面子步骤逐条修改并运行
   - 运行模板验证正常请求。
   - 构造非法 JSON 请求。
   - 补 2 条失败样例。
4. 每完成一步，至少再运行一次确保无回归
## 常见错误与排查
- 解码失败后继续执行。
- 未处理空字段。
- 若报错看不懂，先精简到最小复现场景再定位

## 参考答案要点（不是唯一答案）
- 函数边界清晰：输入/输出/错误语义明确
- 至少覆盖一个边界输入（空、零值、越界、非法）
- 输出内容可解释，不依赖“碰巧正确”

## 自测题（完成后自查）

2. 如果把某一步删掉，会出现什么错误或风险？
