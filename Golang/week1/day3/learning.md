# Week1 Day3 学习内容（超详细指导）

## 今日主题
- 函数、多返回值、defer

## 学前准备
- 会写基本函数。
- 工作目录：`Golang/week1/day3`
- 先执行：`go run template.go`

## 分钟级安排（建议 120 分钟）
- 0-15 分钟：快速阅读模板，标记看不懂的点
- 15-40 分钟：学习核心知识，写下 3 条笔记
- 40-85 分钟：按步骤改代码并验证输出
- 85-105 分钟：做扩展练习
- 105-120 分钟：复盘和整理提交物

## 核心知识讲解
- Go 推荐 `value, error` 返回风格。
- `defer` 在返回前执行，且后进先出。
- 错误信息要包含上下文。

## 手把手实操（命令级）
1. 打开当前目录下 `template.go`
2. 运行：`go run template.go`
3. 按下面子步骤逐条修改并运行
   - 运行模板，观察正常与异常输出。
   - 新增两个 defer 验证执行顺序。
   - 实现 `safeMod(a,b)` 并处理除零。
4. 每完成一步，至少再运行一次确保无回归
## 常见错误与排查
- `defer` 参数在声明时求值。
- 只打印错误不返回会丢上下文。
- 若报错看不懂，先精简到最小复现场景再定位

## 参考答案要点（不是唯一答案）
- 函数边界清晰：输入/输出/错误语义明确
- 至少覆盖一个边界输入（空、零值、越界、非法）
- 输出内容可解释，不依赖“碰巧正确”

## 自测题（完成后自查）

4. 你能给出一个更健壮的版本吗？（例如加入校验/超时/日志）
