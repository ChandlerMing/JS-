## 核心知识讲解
- `string` 是 UTF-8 字节序列。
- 中文处理应按 rune。
- `len(s)` 是字节数，不是字符数。

## 手把手实操（命令级）

2. 运行：`go run template.go`
3. 按下面子步骤逐条修改并运行

   - 打印 `len` 与 `utf8.RuneCountInString`。

## 常见错误与排查
- 按 byte 下标处理中文会乱码。
- 错误地把字符数当字节数。
- 若报错看不懂，先精简到最小复现场景再定位
